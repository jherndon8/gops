Strategy parameters

bidsMade: An array of arrays of length 2. bidsMade[n][0] is the opponent's bid on turn n and bidsMade[n][1] is the treasure value of that round

bidsLeft: An array of numbers 1 - 13 with the already bid cards excluded. The return value must be a value within this array

treasure: the current treasure card

curTie: If there was a tie last round, this variable stores the total value of all tied treasure cards up for grabs. Useful for non-deterministic strategies
